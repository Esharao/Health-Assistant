Healthcare Assistant Chatbot 🏥🤖
A simple AI-powered chatbot built using Streamlit, Transformers (DistilGPT-2), and NLTK to provide healthcare-related responses.

Features 🌟
✔️ Understands basic healthcare queries (symptoms, medication, appointments).
✔️ AI-powered text generation for general responses.
✔️ Interactive web-based UI using Streamlit.
✔️ Fast and lightweight NLP model.

Tech Stack 🛠️
1.Python,
2.Streamlit (for UI),
3.Transformers (DistilGPT-2) (for AI-generated responses),
4.NLTK (for text processing).
